## Start
1. 해당 파일과 **같은 위치**에 `token.txt`, `channel.txt`라는 파일을 만들어 주세요.
2. `token.txt`에 자신의 **Discord bot Token**을 입력해 주세요.
3. `channel.txt`에 사용할 수 있게 할 **체널 ID**를 입력해 주세요. (1개만 입력)
4. Python파일을 실행하시면 됩니다.
